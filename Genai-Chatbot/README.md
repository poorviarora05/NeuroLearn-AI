# NeuroLearn AI 🤖

NeuroLearn AI is a modern AI-powered chatbot designed to help students learn Artificial Intelligence, Machine Learning, LLMs, and Generative AI through interactive conversations.

---

## Features

- Modern iPhone-style UI
- AI/ML learning assistant
- Interactive chatbot interface
- Chat history
- Clear chat option
- Typing animation
- Responsive design
- AI-generated responses
- Quick topic buttons

---

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenRouter API / LLM API
